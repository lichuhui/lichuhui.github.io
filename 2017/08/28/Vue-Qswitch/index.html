Vue-Qswitch

2017-08-28 Vue Vue Component Vue Qswitch 前端 UI Vue

Qswitch

一个帅气的Vue开关组件。

Qswitch项目地址

使用

第一步：注册Qswitch组件

import Qswitch from 'path/to/Qswitch'
Vue.use(Qwsitch)

第二步：添加组件代码片段

为了实现双向绑定，必须设置v-model属性。

<qswitch v-model="ref_to_checked" :value="val"></qswitch>

第三步：使用组件

  import Qswitch from './components/Qswitch.vue'

  export default {
    name: 'app',
    data() {
      return {
        msg: 'An awesome switch component for Vue.',
        val: 'I am value',
        ref_to_checked: false
      }
    },
    components: {
      'qswitch': Qswitch
    }
  }
